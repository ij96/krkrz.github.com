Kirikiri Z

About Kirikiri Z

Kirikiri Z is the follow-up version of Kirikiri 2.
Kirikiri Z can be used to make 2D games and applications.
It has been used in many commercial ADV/visual novels.
It has also been used in the application e-mote/Emofuri.
It has many uses, not limited to games.

KAG3 is an ADV/visual novel development system that uses Kirikiri Z.
krwidgets is a widget toolkit for making GUI applications that uses Kirikiri Z.

Information on Kirikiri 2 can be found in the middle of this page.

Operating Environment
  • CPU : Pentium III 800MHz or above ( 1GHz or above recommended )
  • OS : Windows Vista, 7, 8, 10
  • メモリ : 128MB or above ( 1GB or above recommended )
  • VIDEO : 65536 colours (TrueColor, 16bit) or above ( 16777216 colours (FullColor, 32bit) recommended )
  • DirectX 9 or later required
  • For playing WMV: Windows Media Player 9 or later required
  • For playing H.264: Windows 7 or later required

The operating environment changes depending on the functionalities that are used. It is not the same as saying that all software made by using Kirikiri Z will work pratically in this environment.
Please do not copy and paste the above operating environment - check the software maker's site to confirm and set the right environment for usage.

※Although the content above is not too different compared to that of Kirikiri 2 - apart from the OS, the other conditions are basically the same. However, when taking consideration of the current situation, Kirikiri 2's operating environment is too low, so here the environment settings have been raised higher.
